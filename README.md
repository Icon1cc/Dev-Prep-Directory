## Study Paths

### Path 1: DSA Focus (4-6 weeks)
1. Programming-Languages/Python-Refresher
2. Practice (Mathematics → Queue, in order)
3. OOP fundamentals and patterns
4. LeetCode-Prep daily plans

### Path 2: ML Engineer (8-12 weeks)
1. DSA basics (2 weeks)
2. Machine-Learning full track (6-8 weeks)
3. System-Design ML-specific sections (2 weeks)

### Path 3: Full-Stack Prep (12-16 weeks)
1. DSA complete (4-5 weeks)
2. System-Design (3-4 weeks)
3. Machine-Learning (4-6 weeks)
