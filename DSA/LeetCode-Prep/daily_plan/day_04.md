# Day 4: Tree Fundamentals

## Topics Covered
- Tree traversal basics
- DFS (Depth-First Search)
- Tree comparison
- Binary Search Tree properties

## Problems to Solve

1. **Same Tree** (Easy) - https://leetcode.com/problems/same-tree/
   - Pattern: Recursion
   - **Hint**: Compare recursively at each node

2. **Symmetric Tree** (Easy) - https://leetcode.com/problems/symmetric-tree/
   - Pattern: Recursion
   - **Hint**: Check mirror property recursively

3. **Maximum Depth of Binary Tree** (Easy) - https://leetcode.com/problems/maximum-depth-of-binary-tree/
   - Pattern: DFS
   - **Hint**: Recursively find max depth

4. **Invert Binary Tree** (Easy) - https://leetcode.com/problems/invert-binary-tree/
   - Pattern: Recursion
   - **Hint**: Swap left and right recursively

5. **Lowest Common Ancestor of BST** (Easy) - https://leetcode.com/problems/lowest-common-ancestor-of-a-binary-search-tree/
   - Pattern: BST Property
   - **Hint**: Use BST property to guide search

6. **Binary Tree Paths** (Easy) - https://leetcode.com/problems/binary-tree-paths/
   - Pattern: DFS
   - **Hint**: DFS with path tracking

7. **Minimum Depth of Binary Tree** (Easy) - https://leetcode.com/problems/minimum-depth-of-binary-tree/
   - Pattern: DFS
   - **Hint**: Find leaf node with minimum depth

## Key Concepts Learned Today

✅ **Tree Recursion Pattern**
- Base case: null node
- Recursive case: process left and right subtrees
- Return combined result

✅ **Binary Search Tree Property**
- Left subtree < node < Right subtree
- Enables efficient navigation
