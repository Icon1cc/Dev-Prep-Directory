## Study Order

### Week 1-2: Foundations
1. **Python-Refresher** - Language basics, OOP, decorators
2. **Mathematics** - GCD, LCM, primes, factorization

### Week 3-4: Core Data Structures
3. **Lists** - Array operations and algorithms
4. **Recursion** - Recursive thinking patterns
5. **Searching** - Binary search and variations
6. **Sorting** - All major sorting algorithms

### Week 5-6: Advanced Structures
7. **Hashing** - Hash tables, collision handling
8. **Strings** - Pattern matching, manipulation
9. **Linked Lists** - Singly linked list operations
10. **Circular & Doubly Linked Lists**

### Week 7-8: Abstract Data Types
11. **Stack** - Implementation and applications
12. **Queue** - Various implementations

### Week 9-10: Interview Prep
13. **OOP** - 100 problems covering patterns & SOLID
14. **LeetCode-Prep** - Daily practice plans
